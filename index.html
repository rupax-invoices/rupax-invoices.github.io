Enterprise-grade security

Your data is protected
at every layer

Security is not a feature — it's the architecture. Every invoice goes through multiple layers of cryptographic protection.

AES-Fernet Encryption
Invoice payloads are encrypted at rest using symmetric AES-128 in CBC mode via the Fernet scheme before database storage.
HMAC Integrity Check
Each payload is signed with HMAC-SHA256. Any modification to an invoice in storage is detected before the PDF is served — zero silent corruption.
Hashed Email Storage
No email addresses are stored in plaintext. Your email is SHA-256 hashed before it ever touches the database — making user re-identification impossible.
Automatic Expiry
Invoices and their PDFs are automatically purged after 7 days. Stale data is never retained. Tokens become invalid and PDF files are removed from the server.
